山葡萄編みのかごバッグ 遠野 | 公式サイト制作プロジェクト
岩手県遠野市で活動する伝統工芸作家（実父）の作品を紹介・販売するための公式Webサイトです。
「一生もの」としての品質、遠野の厳しい自然が育む素材の力強さ、そして手仕事の繊細さを視覚的に伝えることを目的として制作しました。

✨ 実装のこだわりと技術的ポイント
1. 「伝統×モダン」を体現するビジュアル設計
レスポンシブ・ヒーローセクション: - デバイスサイズに応じて、表示する画像の枚数とレイアウトを最適化（PC: 4枚並列 / Tablet: 2枚 / Mobile: 上下2枚）。
どの環境でも迫力ある視覚体験を提供します。

独自のデザイン要素:ギャラリーセクションでは、あえて画像を正円に切り抜き、境界線（border）を付与。伝統工芸のサイトにありがちな「堅苦しさ」を払拭し、現代のライフスタイルに馴染む軽やかさを演出しました。
backdrop-blur-md（背景ぼかし）を活用したモバイルメニューにより、高級感を維持したまま視認性を確保しています。

2. コンバージョンを意識したUI/UX
導線の最適化: - 直接注文（Instagram DM）と、手軽な購入（メルカリ）の2つのルートを提示。ターゲットユーザーのITリテラシーや購入スタイルに合わせた選択肢を用意しました。

マイクロインタラクション:ホバー時の画像拡大（scale-110）や、スクロールに合わせたフェードイン（Intersection Observer）により、サイト全体の質感を高め、「丁寧な仕事」というブランドイメージを補強しています。

3. Tailwind CSSによる効率的なスタイリング
カスタムカラー（brand-navy, brand-gold等）を定義し、サイト全体で一貫した色彩設計を維持。

インラインでの記述を活かしつつ、保守性を考慮して共通のフェードインアニメーションなどはCSSクラスとして外部定義し、コードの肥大化を防いでいます。
